Install & Run Instructions:

1. Download & Setup Ollama: https://ollama.com/download
2. Run Ollama
3. Download a model on the command line using "ollama pull qwen3-vl:4b"
4. Download ollama into your python environment usint "pip install ollama"
5. (Optional) Edit the query in "content" of line 55 in minimal_example.py
6. Run the query with "python minimal_example.py"
